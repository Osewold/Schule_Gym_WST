# Schule_Gym_WST

Hallo! Dieses Repository dient der Verwaltung und Planung schulischer Aufgaben am Gymnasium WST.

## Inhalt

Dieses Repository wird genutzt für:

- **Stundenplan-Revisionen** – Planung und Dokumentation von Stundenplanänderungen
- **Personalplanung** – Elternzeiten, Rückkehr von Lehrkräften, Vertretungsregelungen
- **Referendariat** – Begleitung und Datenbankpflege für Referendar:innen
- **Fortbildungen** – z. B. KI-Selbstlernkurse (Land NDS)

## Nutzung

Issues werden verwendet, um Planungsaufgaben zu dokumentieren und nachzuverfolgen. Jede Revision oder Änderung am Stundenplan wird als Issue angelegt und kommentiert.
